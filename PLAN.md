# 구현 계획 (PLAN)

## 주차별 마일스톤

### 1주차 — 센서 및 입력부 검증
- RC522 RFID 리더 단독 동작 (UID 읽기)
- I2C LCD 16x2 출력 테스트
- 로터리 엔코더 회전/버튼 입력 처리
- IR 센서 거리 감지 캘리브레이션

### 2주차 — 액추에이터 및 게이트부
- 릴레이 모듈 ON/OFF
- 서보모터 각도 제어 (게이트 0°/90°)
- 부저 알람 패턴 정의 (성공음/경고음)

### 3주차 — 통합 및 데이터 흐름
- EEPROM 상품 DB 구조 설계 및 초기화
- 6단계 상태머신(scan → cart → total → pay → gate → open) 통합
- LCD UI 시퀀스 흐름 완성

### 4주차 — 테스트 및 수정
- 정상 결제 시나리오 회귀 테스트
- 미결제 통과 시 부저 알람 동작 확인
- 데모용 상품 5종 NTAG213 스티커 부착
- 발표 자료 작성

---

## 변경 로그
계획이 변경되거나 구현이 막혔을 때 여기 기록한다. (밀린 일기 형식, 시간은 적지 않음)

### 초기 계획 수립
- 위 4주 계획으로 시작.
